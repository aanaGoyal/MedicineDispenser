Health Access Made Simple and Secure.

The IoT Smart First-Aid Kit automates medicine dispensing using a secure, web-based system. Students can log in to request medicines, which are then precisely dispensed from the kit, ensuring safety and accountability.

Get Started

Key Features of Our System

Secure User Login

Only registered users with a valid student ID can log in and access the system to make medicine requests.

Web-Based Request System

Users can select and request medicines conveniently from any web browser, streamlining the process.

IoT Controlled Access

Your request is sent to the IoT dispenser, which automatically unlocks the correct compartment for the selected medicine.

Real-Time Alerts

Receive instant notifications for low stock, expired medicines, and successful dispensing events directly from the dashboard.

Complete Transaction Logs

Every medicine request and dispensing action is recorded in the database, providing a full audit trail for administrators.

Enhanced Safety

By automating access and dispensing, the system prevents unauthorized use and reduces the risk of human error.
